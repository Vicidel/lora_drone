These commands launch MAVROS on the RPi and links it with the Pixhawk
Then we launch the C++ script offb_node_simple
Can spectate from QGC on another computer using 433 telemetry

	Terminal 1 (on RPi via SSH):
		// connects the RPi with the Pixhawk
		roslaunch mavros px4.launch fcu_url:=serial:///dev/ttyS0

	Terminal 2 (on RPi via SSH):
		// starts the offboard node
		cd catkin_ws
		catkin build vic_package
		source devel/setup.bash
		rosrun vic_package offb_node_simple
